About Us

This is a data-driven website example for University of London's Responsive Web Design MOOC class offered on Coursera. The course is the 2nd in the required sequence for achieving the Responsive Website Development and Design specialization.

Requirements for this assignment iteration include:

  1. Create at least 1 HTML file that contains the structure of our site, including templates designed to display that data.
  2. Create a JavaScript file that maps our data onto the templates and provides page interactivity.
  3. Create a CSS file that styles our page, including optional use of the Bootstrap library.
  4. The overall layout should clearly display data in an understandable way, and ensure that the end-user always knows where they are, where they can go, and what they can do.
